Current approach: A horizontal slider with 5 policy positions like "Universal support programs" ‚Üí "Minimal safety net". The user drags a thumb along a gradient bar.

What changes below: Questions are reframed around values and principles. Users tap cards instead of dragging sliders. All three approaches now include a "How strongly?" step that separates direction (what you believe) from intensity (how firmly you hold it). This two-step pattern captures both position and conviction in a clean, tap-friendly way.
