Value-Framed Explanation Language
Uses Schwartz value theory to generate resonant, value-grounded language when explaining why ballot measures or candidates align or conflict with a user's civic blueprint. Copy/language enhancement only — no changes to scoring or data model.
How it works
Meta-dimensions → Value poles: Each user's 3 meta-dimension scores map to a dominant pole. Community-leaning users get "collective wellbeing" framing; individual-leaning get "personal responsibility" framing.
Policy → Resonance/Tension: When a ballot measure aligns with the user's dominant pole, show a resonance phrase. When it conflicts, show a tension phrase. Neutral dimensions (|score| < 0.15) are skipped.
