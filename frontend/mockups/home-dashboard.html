Feature Breakdown

Key components across all designs

1. Tab Status Cards

  • Smart Assessment: Shows completion status, questions answered, and confidence score
  • Blueprint: Shows domains configured, fine-tuning progress
  • Ballot Builder: Shows contests decided, measures reviewed

2. Election Countdown

  • Days/hours/minutes until Election Day
  • Election name and type (Primary, General, Special)
  • Registration deadline warning
  • Early voting dates

3. Poll Locator

  • Address input or GPS location
  • Nearest polling place with address
  • Operating hours
  • Directions link (Google Maps integration)
  • Early voting location option

API Options for Poll Locator

  • Google Civic Information API: Free, reliable, official data
  • Vote.org API: Nonprofit, election-focused
  • State Election Board APIs: Direct from source, varies by state
